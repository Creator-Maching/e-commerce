# 🛒✨ MTG E-Commerce — Seu Marketplace de Cartas Magic

Bem-vindo ao seu projeto de e-commerce de cartas de **Magic: The Gathering** ⚡
Este não é apenas mais um projeto — é a materialização do seu aprendizado em **frontend + backend thinking**, integrando APIs reais, estado global e experiência de usuário.

---

## 🚀 Visão do Projeto

Uma aplicação moderna construída com foco em:

🧠 Aprendizado consciente
⚙️ Arquitetura escalável
🎯 Experiência fluida para o usuário

O sistema permite buscar cartas, visualizar detalhes e montar um carrinho de compras funcional — tudo com persistência e integração externa.

---

## 🧩 Funcionalidades

🔍 **Busca de cartas (API Scryfall)**

* Pesquisa dinâmica
* Tratamento de cartas multi-face
* Exibição com preview em alta resolução

🃏 **Visualização de cartas**

* Hover com zoom 🔎
* Informações detalhadas
* Layout responsivo

🛍️ **Carrinho de compras**

* Adição e remoção de itens
* Controle de quantidade ➕➖
* Persistência com `localStorage` 💾

💱 **Conversão de preços**

* Integração com API de câmbio
* Valores exibidos em BRL 🇧🇷

📱 **Responsividade**

* Funciona em desktop e mobile
* Layout adaptável

---

## 🧱 Tecnologias Utilizadas

⚛️ **React + TypeScript**
⚡ **Next.js**
🎨 **Tailwind CSS**
🌐 **Scryfall API**
💱 **Exchange Rate API**
🧠 **React Context (estado global)**
💾 **localStorage (persistência)**

---

## 🧠 Arquitetura (como você pensou o projeto)

* Separação clara de responsabilidades
* Componentização reutilizável
* Estado global desacoplado
* Consumo de APIs externas tratado de forma isolada

Esse projeto não foi só “feito” — ele foi **entendido**.

---

## 📦 Estrutura Simplificada

```
/src
  /components   → UI reutilizável
  /context      → estado global (carrinho)
  /services     → integração com APIs
  /pages        → rotas da aplicação
  /types        → tipagem TypeScript
```

---

## ⚙️ Como rodar o projeto

```bash
# Instalar dependências
npm install

# Rodar ambiente de desenvolvimento
npm run dev

# Build de produção
npm run build
```
